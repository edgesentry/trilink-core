# Tri-Link

Tri-Link bridges a patrol robot platform and an AI damage detection service so that customers can open the site's 3D model and see exactly where vehicle damage is located — which car, which surface, what kind of damage.

Neither the robot platform nor the AI platform connects the image to a 3D location. That is Tri-Link's sole job. See [docs/assumptions.md](docs/assumptions.md) for the full role breakdown and [docs/architecture.md](docs/architecture.md) for how the data flows.
